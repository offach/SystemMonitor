# SystemMonitor
Приложение для мониторинга ограниченного перечня параметров компьютера.

Это приложение предоставляет возможность мониторинга системных метрик в реальном времени, включая температуру GPU, использование CPU, использование GPU, использование оперативной памяти и использование диска.

### Возможности
- Мониторинг температуры GPU (только Windows/Linux с NVIDIA GPU)
- Мониторинг использования CPU
- Мониторинг использования GPU (только Windows/Linux с NVIDIA GPU)
- Мониторинг использования оперативной памяти
- Мониторинг использования диска
- Настройка отображаемых метрик
- Обновление данных в реальном времени каждые 5 секунд

## Требования

- Python 3.7 или выше
- tkinter (обычно входит в стандартную установку Python)
- psutil
- GPUtil (опционально, только для Windows/Linux с NVIDIA GPU)

## Установка зависимостей

```bash
pip install -r requirements.txt
```

## Сборка приложения

### Для macOS

Запустите скрипт сборки:

```bash
./build_mac.sh
```

Собранное приложение будет находиться в папке `dist/SystemMonitor.app`

### Для Windows

Запустите скрипт сборки:

```cmd
build_windows.bat
```

Собранный исполняемый файл будет находиться в папке `dist/SystemMonitor.exe`

## Запуск из исходников

```bash
python main.py
```

## Примечания

- **GPU метрики**: Работают только на Windows/Linux с NVIDIA GPU. На macOS будут отображаться как "N/A (Mac/No GPU)"
- **Диск**: На Windows мониторится диск C:\, на macOS и Linux - корневой раздел (/)
